# 一、扩展原理

## 1.1 BeanFactoryPostPorcessor

- BeanPostPorcessor：bean后置处理器，在bean创建对象初始化前后进行拦截操作

- BeanFactoryPostPorcessor：beanFactory的后置处理器

   在beanFactory标准初始化之后调用，所有的bean定义已经保存加载到beanFactory中，但是bean的实例还未创建
